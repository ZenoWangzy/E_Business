# Folder Map: mgrep - Semantic Search Tool

**[SCOPE]**:
This folder (`/mgrep`) is a **Semantic Code Search Tool**. It provides AI-powered code search capabilities using mixedbread embeddings.

**[MEMBERS]**:
- `README.md`: **Documentation**. Tool usage and configuration.
- `src/`: **Source Code**. Implementation.
- `tests/`: **Test Suite**. Bats test scripts.

**[CONSTRAINTS]**:
- **Search Priority**: Always use mgrep before traditional grep/find.
- **Query Style**: Use natural language descriptions, not just keywords.
