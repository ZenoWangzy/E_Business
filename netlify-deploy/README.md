# 电商详情页生成工具

一个帮助电商卖家快速生成商品详情页和轮播图的在线工具。

## 功能特点

### 📤 智能素材上传
- 上传商品白底图
- 支持多格式文件读取：PDF、Word、Excel、图片
- 自动提取文档内容（产品说明书、工厂信息）

### 🏷️ 品类管理
- 52个商品品类选择
- 支持搜索快速定位

### 🎨 风格模板
- 6种详情页风格模板
- 现代简约、高端奢华、清新自然、科技感、温馨风格、商务专业

### 🖼️ 图片生成与管理
- 自动生成5-8张轮播图和10-13张详情页
- 每张图独特配色，高辨识度
- 图片序号标识

### 🎯 高级编辑功能
- **360度拖拽**：支持上下左右任意方向调整顺序
- **批量删除**：复选框多选删除
- **单张删除**：快速删除单张图片
- **上传参照图**：每张图可上传参考图片
- **文字标注**：每张图可添加说明文字

### 👁️ 预览功能
- **轮播图预览**：左右滑动查看，支持键盘控制
- **详情页预览**：合并长图预览
- **下载功能**：分别下载或下载合并长图

### 💾 导出下载
- 分别下载轮播图和详情页
- 下载合并长图
- 下载单张图片

## 快速开始

### 本地运行

1. 克隆或下载项目到本地
2. 使用任意Web服务器运行，例如：

**使用Python:**
```bash
python -m http.server 8000
```

**使用Node.js (http-server):**
```bash
npx http-server -p 8000
```

**或直接打开:**
```bash
# 直接在浏览器中打开 index.html
```

3. 访问 `http://localhost:8000`

## 部署方式

### 1. GitHub Pages (推荐)

1. 将代码推送到GitHub仓库
2. 进入仓库设置 Settings > Pages
3. 选择分支（通常是main）
4. 点击Save，等待部署完成
5. 访问 `https://你的用户名.github.io/仓库名`

### 2. Vercel

1. 访问 [vercel.com](https://vercel.com)
2. 导入GitHub仓库
3. 自动部署完成

### 3. Netlify

1. 访问 [netlify.com](https://netlify.com)
2. 拖拽项目文件夹到Netlify
3. 自动部署完成

### 4. 服务器部署

将所有文件上传到服务器的Web目录（如 `/var/www/html`）即可。

## 使用说明

### 步骤1：上传素材
- 上传商品白底图（必需）
- 上传产品说明书（可选）
- 上传工厂信息：证书、流水线图、商标、质量认证（可选）

### 步骤2：选择品类
- 从52个品类中选择商品类别
- 支持搜索快速定位

### 步骤3：选择风格
- 选择适合的详情页风格模板
- 6种风格：现代简约、高端奢华、清新自然、科技感、温馨风格、商务专业

### 步骤4：生成详情页
- 自动生成轮播图（5-8张）
- 自动生成详情页（10-13张）
- 调整图片顺序：拖拽或使用↑↓按钮
- 分别下载轮播图和详情页

## 技术栈

- **前端框架**：原生 HTML + CSS + JavaScript
- **文件读取**：
  - PDF.js - PDF文件读取
  - Mammoth.js - Word文档读取
  - SheetJS (XLSX) - Excel文件读取
- **图片处理**：Canvas API
- **无需后端服务器**
- **无需数据库**
- **响应式设计**
